# API-Automation-Test-
API Automation Test Implement with Model, View, Controller

Preparation for running test using katalon studio v9.5.0
- Install Katalon Studio 
- Install nodejs
- Install jdk 21.0.2

How to running test :
- clone project form git
- Open project using katalon studio
- Click folder Object Repository
- Click GetPersons
- CLick Execute
- View Response

How to running using test suites :
- Click folder test suites
- Click APIpersons
- Click Run or Ctrl + Shift + A
